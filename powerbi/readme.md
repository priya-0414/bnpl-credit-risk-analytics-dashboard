# Power BI Dashboard

This folder contains the Power BI dashboard developed for the **BNPL Credit Risk Analytics Project**.

## File

### bnpl_risk_dashboard.pbix

This Power BI file contains an interactive dashboard designed to analyze and monitor **Buy Now Pay Later (BNPL) credit risk**.

The dashboard provides insights into loan portfolio performance, borrower risk profiles, and expected financial losses.

---

## Dashboard Pages

### 1. Portfolio Overview

Provides a high-level summary of the loan portfolio.

Key metrics include:

- Total Portfolio Exposure
- Portfolio Default Rate
- Total Expected Loss
- Loan Approval Rate
- Risk Distribution across categories

This page helps stakeholders quickly assess the **overall health of the credit portfolio**.

---

### 2. Risk Intelligence

Analyzes factors influencing credit risk.

Visuals include:

- Default Rate by Risk Category
- Risk Score Distribution
- Income Segment vs Risk Category

This page helps identify **risk drivers within the borrower population**.

---

### 3. Portfolio Monitoring

Tracks financial exposure across risk segments.

Visuals include:

- Loan Exposure by Risk Category
- Expected Loss by Risk Category
- Loan-level portfolio table

This page supports **continuous monitoring of risk concentration and financial exposure**.

---

### 4. Customer Risk Profile

Provides drillthrough analysis for **individual borrower evaluation**.

Displays:

- Customer income
- Loan amount
- Risk score
- Expected loss
- Credit decision
- Loan vs Income comparison

This allows analysts to assess **borrower-level credit risk**.

---

### 5. Credit Policy Simulation

An interactive scenario analysis tool used to evaluate **credit approval strategies**.

Features include:

- Risk Threshold Slider
- Approved vs Rejected Loans
- Expected Portfolio Loss under current policy
- Loan approval distribution by risk band

This page demonstrates how **policy changes impact portfolio risk and loan approvals**.

---

## Key Metrics Implemented

The dashboard uses **DAX measures** to calculate important portfolio metrics:

- Total Exposure
- Portfolio Default Rate
- Expected Loss
- Approval Rate
- Risk Concentration
- Simulated Loan Approval
- Simulated Portfolio Loss

---

## Purpose

This dashboard demonstrates how data analytics can support **credit risk monitoring, portfolio management, and policy decision-making in fintech lending platforms**.
